grid-fraction.less

Why?

  1. Make on top of Twitter Bootstrap Scaffolding
  2. Save time on hacking
  3. No gutter
  4. Flexible layout with more classes
  5. Responsive layout with percentage
  6. As simple as possible

Basic HTML Structure

  1. Just like using Twitter Bootstrap Scaffolding
  2. Add .row-fraction to .row
  3. Use fraction column class instead of .span*, e.g. .full, .half, .third*, .quarter*, .fifth*, .sixth*
  4. Add .inner to your inner content for .row-padding, .row-margin, .row-balance, .row-border, .row-phone, .row-tablet
<div class="container">
    <div class="row row-fraction"> 
        <div class="full">
            <div class="inner">...</div> 
        </div> 
    </div> 
</div>
